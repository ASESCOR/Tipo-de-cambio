Las cotizaciones actuales se consultan desde DolarApi. El histórico diario se guarda en data/history.json mediante GitHub Actions. Las reservas, el tipo de cambio real y la base monetaria continúan como series manuales de ejemplo hasta conectarlas con una fuente oficial.
